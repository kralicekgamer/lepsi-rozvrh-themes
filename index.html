Lepší rozvrh – motivy a compiler

Tento projekt obsahuje vlastní motivy pro aplikaci Lepší rozvrh a nástroj pro jejich kompilaci.

Jak to funguje

Aplikace Lepší rozvrh umožňuje import motivů, ale nepodporuje přímé zadávání HEX barev. Místo toho se motiv zadává jako ID řetězec — což je zakódovaný JSON.

JSON se nejprve zkomprimuje pomocí GZIP, a poté zakóduje do Base64. Tento proces zajišťuje skript compiler.py nebo alternativně CyberChef.
